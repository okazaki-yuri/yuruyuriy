# 変更履歴

本プロジェクトの主な変更点を記録する。フォーマットは [Keep a Changelog](https://keepachangelog.com/ja/1.1.0/) に準拠する。
日付は YYYY-MM-DD（JST）。

## 2026-07-23
### 追加
- **英語版ページ（`/en/`）を公開**。トップ・ツール一覧・単語ルーレット（Word Roulette）・オンラインサイコロ（Web Dice）・お問い合わせ・利用規約・プライバシーポリシーの全ページを英語で提供。日本語版の URL は従来どおり変更なし。
- 全ページのヘッダーに**言語スイッチャー**（日本語ページには「English」、英語ページには「日本語」）を追加。現在のページに対応する言語版へ相互リンクする。
- 英語版の利用規約・プライバシーポリシーに **「日本語版が正文であり、相違がある場合は日本語版が優先する」旨の Language 条項**を明記。

### 変更
- サイト名の英語表記を「Ylyuriy」に決定（© 表記・ドメイン・SNS ID と統一）。ツールの英語名は「〜ちゃん」を付けないシンプルな命名（Word Roulette / Web Dice）とした。

## 2026-07-22
### 追加
- 単語ルーレットに**円形ルーレット（ホイール式）**を追加。「テキスト式／ホイール式」をタブで切り替えでき、選択は次回アクセス時も保持される。ホイールは当選のことばが上部ポインターで止まるよう回転し、抽選時間の設定に連動。OS の「視差効果を減らす」設定時はアニメーションを省略する。

### 変更
- SEO: サイトの説明文言を「**かわいくて便利な無料Webツール**」の軸に統一（デフォルト title・トップの description・OGP・JSON-LD）。不自然だった「自主制作」「個人製作」は「個人開発」に変更。
- SEO: 内容の薄かった meta description を具体化（ツール一覧・お問い合わせ・プライバシーポリシー・利用規約。OGP description も同時に更新）。ツール一覧カードの WEBサイコロちゃんの説明（`data/tools.json`）も「サイコロツールです。」から機能のわかる説明文へ変更。
- SEO: OGP タイトルを「ページ名 | ゆるユーリ」の語順に統一し、title タグと一致させた。ツールページの title の区切りを全角「｜」から「（）」に変更し、template の「 | 」との区切り文字混在を解消。
- 表記統一: 半角チルダ「~」を全角波ダッシュ「〜」に統一（WEBサイコロちゃんの使い方ガイド・入力フォームのラベル／補足文）。
- 文章改善: 「ボタン押下」「選出します」「その他詳細なご利用については」などの硬い・不自然な表現をやわらかい表現に修正。WEBサイコロちゃんの使い方ガイドを**手順（番号付きリスト）と補足事項に分離**し、ツール概要の統計名称を UI 表示（合計・平均・最大・最小）に統一。お問い合わせの案内文の矛盾（フォーム必須と読める文言）を解消。利用規約の「不正な転載や商標利用」を「無断での転載や再配布」に具体化し、規約の更新日を 2026-07-22 に更新。
- デザイントークン（CSS カスタムプロパティ）を導入し、色・タイポグラフィ・余白・コンテナ幅を `:root` で一元管理化。個別 CSS は生の値を直書きせず `var(--token)` を参照する方針に統一。
- 配色を**グレー×クリームのくすみカラー**へ刷新（主アクセント: ダスティローズ `#7f4f56` / 副アクセント: スモーキーセージ）。従来のオレンジ・黒・青の混在を解消し、主要な文字色・UI色を WCAG AA 以上のコントラストに調整。
- フォントのウェイト `400;500;700` を明示読み込みし、見出し・ボタンの**合成ボールド（faux bold）を解消**。日本語システムフォントをフォールバックに追加。
- タイプスケール・行間をトークン化し、サイズ指定の単位を `rem` に統一。
- コンテナ戦略を見直し。`body` の `max-width` を撤去し、**ヘッダー／フッターを全幅背景＋中身中央寄せ**に。コンテンツ幅・余白をトークン化して各ページの基準を統一。
- WEBサイコロちゃんの入力エリア・サイコロ表示エリアを中央寄せに修正（3エリアの横幅を統一。入力エリア内の文言・入力欄は左寄せを維持）。
- 本文の既定を左寄せに変更し、中央寄せは各要素で明示する方針に統一（長文の可読性向上）。
- ツール一覧の「Coming Soon」プレースホルダーカードの重複マークアップを配列生成に整理。

### 修正
- フッターの装飾フェード線（`::before` / `::after`）の位置基準が無かった不具合を修正（`.site-footer` に `position: relative`）。
- CSS のスペース抜け（`border: 2px solid#555` 等）・重複指定・不要なフォント上書きを整理。

### ドキュメント
- `docs/style-guide.md` の配色・タイポグラフィ・レイアウト方針を、デザイントークンに基づく内容へ全面更新。
